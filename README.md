# Tableau Specialization

During this specialization, I worked with two fictional datasets: E-commerce and Superstore.
- E-commerce Dataset: E-commerce Funnel Chart
- Superstore Dataset:
  - Tables, Bar Charts, Pie Charts, Line Charts, Treemaps, Bubble Charts, Highlight Tables, Heat Maps, Area Charts, Maps, Symbol Maps, Choropleth Maps, Choropleth Maps with custom territories using grouping, Density Maps, Dual-Axis Maps (e.g., A Symbol Map stacked on top of a Choropleth Map)
  - Motion Charts, Dual-Axis Charts (e.g., A Line Chart stacked on top of a Bar Chart), Gantt Charts, Profit & Sales by Category using a Bar-in-Bar Chart, Sales by Category using a Donut Chart, Quantity Breakdown by Sub-Category using a Waterfall Chart, Monthly Sales by Category and Segment using Sparkline Charts, Map Layers
  - Data Source Filters, Worksheet Filters, Context Filters, Folders, Hierarchies, Drilling Down, Date/Time Hierarchies
  - Custom Groups, Creating Groups and Sets, Aggregate vs Non - Aggregate Functions in Calculated Fields, Quick Table Calculations, Table Calculations, Grand Totals and Subtotals
  - Histograms, Box Plots, Scatter Plots, Reference Line, Reference Band, Distribution Band, Trend Line, Forecast
  - Sales & Profit by Category & Sub-Category, Sales & Profit by Location, Sales & Profit Over Time, Relationship Between Sales & Profit
  - Performance Overview Map, Performance Overview Table, Superstore Performance Overview (Dashboard)
  - Examining Superstore Profit By Location, Regional Profit Overview (Dashboard)
  - Created a Story:
     - Story Point 1 - How are the regions performing?
     - Story Point 2 - How are the states and cities performing?
     - Story Point 3 - How significant are the city gains and losses?
     - Story Point 4 - How have the cities been performing over time?
     - Story Point 5 - Conclusion
